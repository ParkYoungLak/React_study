리액트에 대해서

  • react 는 모바일 어플리케이션과 비슷한 ui를 제공하기 위한 웹 어플리케이션 제작도구
  • react 는 자바스크립트를 기반으로 react-javascript 와 react-DOM을 제작해서 제공
  • react-DOM 은 react-javascript 안에서 JSX 라는 문법으로 html 구문을 작성할 수 있다.
  • 빌드된 react 로 만든 어플은 html css js 파일로 되어 있다.
  • react 에서 함수와 클래스는 컴퍼넌트를 제작하기 위하 객체이고 Component 는 ui를 구성하는 한 부분이다.

react 의 구동 원리

  • react 는 root Component가 여러 컴퍼넌트를 호출하여 react-DOM 을 구성하고 화면에 render할 수 있는 htmlDOM 을 만들고 출력한다. ReactDOM.render()
  • 이때 Component에게 초기값을 매개변수 props 로 줄 수 있다.
  • props 는 수정불가한 초기값으로 컴퍼넌트가 랜더링 할때만 호출되길 기대한다.
  • props 로 화면에 출력된 값은 바뀌지 않는다.
